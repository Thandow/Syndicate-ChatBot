# Syndicate-ChatBot
# Landbot Web Integration Project

## Overview
This project demonstrates multiple ways to integrate a **Landbot chatbot** into a website using **Landbot 3.0**. The chatbot is configured using a hosted Landbot JSON configuration and embedded via JavaScript modules.

The project includes examples of:
- Livechat widget
- Full-page chatbot
- Embedded container chatbot
- Popup chatbot with lazy loading

These approaches allow flexibility depending on UX and performance requirements.

---

## Technologies Used
- HTML5
- JavaScript (ES Modules)
- Landbot 3.0 SDK
- Hosted Landbot configuration (JSON)
